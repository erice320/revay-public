e-automate (ECI) — Migration Data Export Guide

Field-level instructions for pulling your data out of e-automate before onboarding. Send these files to your SA ahead of your kickoff call.

e-automate (ECI Solutions)

Cloud & On-Premise  ·  Import/Export Utility (Excel) + right-click grid export  ·  Est. 45–60 min

⚠️ Before you start: e-automate uses a built-in Data Import/Export Utility. Cloud/hosted users: download export templates from within e-automate (Help → Export Templates) and drag them into the utility window. On-premise users: go to Tools → Import/Export or find the utility as a standalone Excel file. Need help? Call ECI support: 1-800-989-3367.
Customer & Contact Data Source: ECI webhelp.e-automate.com — Data Import Utility docs; Rev.io PSA API /customers & /contacts schemas
1

Customer Accounts

Must Have
How to export
  1. Go to Customers in the left nav → filter to Active
  2. Option A (easiest): Right-click the customer list grid → Export to Excel
  3. Option B (cloud): Open the Import/Export Utility → download Customers Export template → drag into utility → click Export
  4. Save as ea-customers.xlsx

Service Contracts (Recurring Billing)

Must Have
  1. Go to Contracts → filter to Active
  2. Right-click grid → Export to Excel, or use Import/Export Utility with Contracts Export template
  3. Also export Contract Items (line items per contract) as a separate file if available
  4. Save as ea-contracts.xlsx and ea-contract-items.xlsx

GL Codes / Chart of Accounts

Must Have
  1. Go to Accounting → Chart of Accounts (or GL → Accounts)
  2. Right-click the list → Export to Excel
  3. If unavailable: ask your ECI rep to pull a GL Account List report; a PDF printout also works
  4. Save as ea-gl-codes.xlsx
e-automate FieldRev.io PSA FieldNotes

AccountType (Revenue/Expense)→(classification)Rev.io PSA primarily uses revenue GL codes on products
